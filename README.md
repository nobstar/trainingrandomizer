# Training Randomizer

Small (mobile) app for randomized (less boring) workouts.

[Open App](https://nobstar.github.io/trainingrandomizer)

## Features

- Randomized trainings, tool will try to include all body muscle groups/body areas (currently full body workouts only, no specific split training support)
- Tool will prioritize exercises not in last training for less boring workouts
- Allows selecting dumbbell, barbell and/or bodyweight exercises
- Simple statistics (which exercise was done how often), but no support for tracking training dates, used weight or repetition count
- Google links to find videos of the exercises
- Tool is browser based but works fully offline, no server required, no tracking (except for google/youtube search).

## Technologies

Vue 3, TypeScript, Vite
